## Base de Dados: mtcars

### Descrição da base

Dados de desempenho e design de automóveis (extraídos da revista *Motor Trend*, 1974): cilindros, cilindrada, potência, relação de eixo, peso, tempo no quarto de milha, tipo de motor, câmbio, número de marchas e carburadores.

**Variável resposta:**
* **mpg** — consumo de combustível, em milhas por galão (quanto maior, mais econômico).

---

### Dicionário de Variáveis

> Este é um resumo das variáveis. Para a descrição completa, rode no R: `help("mtcars")`

| Variável | Descrição |
|---|---|
| **mpg** | Consumo, em milhas por galão (variável resposta) |
| **cyl** | Número de cilindros |
| **disp** | Cilindrada, em polegadas cúbicas |
| **hp** | Potência bruta, em cavalos |
| **drat** | Relação do eixo traseiro |
| **wt** | Peso, em milhares de libras |
| **qsec** | Tempo no quarto de milha, em segundos |
| **vs** | Formato do motor (0 = em V, 1 = reto) |
| **am** | Tipo de câmbio (0 = automático, 1 = manual) |
| **gear** | Número de marchas à frente |
| **carb** | Número de carburadores |
